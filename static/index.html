1. Ingest Text

Paste arbitrary text and send it to the /upsert endpoint. The server chunks and stores it in your per-user Pinecone namespace.

2. Retrieve Context (optional)

Hit /query to see the nearest chunks this RAG service would use. Helpful when debugging ingest quality.

3. Ask with Streaming

Submit a question to /ask-context-stream. Tokens stream in as they are generated.

Per-user isolation is handled via a persisted localStorage UUID sent as X-User-ID with each request.
