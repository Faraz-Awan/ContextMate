Load documents into Pinecone, then ask streaming questions backed by retrieved context.

1. Ingest Text

Paste arbitrary text and send it to the `/upsert` endpoint. The server chunks and stores it in Pinecone.

2. Retrieve Context (optional)

Hit `/query` to see the nearest chunks the RAG service would use. Helpful when debugging ingest quality.

3. Ask with Streaming

Submit a question to `/ask-context-stream`. Tokens stream in as they are generated.
